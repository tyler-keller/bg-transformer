> [!IMPORTANT]  
> If you want to run the code, you will have to unzip the DiaTrend dataset from [DiaTrend@Synapse](https://www.synapse.org/Synapse:syn38187184/wiki/619490).
> Additionally, you'll have to execute the `clean_data.py` script for your intended subject file(s).
> Script assumes that the DiaTrend dataset has been unzipped into a subdirectory called `data/original/`.
